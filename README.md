# Finance Analyzer Dashboard

An interactive Next.js application for analyzing HDFC bank statements with real-time visualizations.

## Features

- **Drag & Drop Upload** - Simply drop your XLS/XLSX statement file
- **Real-time Analysis** - Instant parsing and categorization of transactions
- **Interactive Charts** - Pie charts for categories, bar charts for daily trends
- **Smart Categorization** - Automatic categorization into 14+ spending categories
- **Merchant Analysis** - See where you spend the most
- **Spending Insights** - AI-generated insights about your spending patterns
- **Transaction Explorer** - Search and filter through all transactions

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## Usage

1. Export your HDFC bank statement as XLS/XLSX
2. Drop the file on the upload area
3. View your spending analysis instantly

## Tech Stack

- Next.js 14
- React 18
- TypeScript
